# Changesets

Versioning for `@agentica-ds/tokens` and `@agentica-ds/components`, independently
versioned per ADR-072. `site/` is excluded (see `ignore` in `config.json`) — it's
private and never published.

Run `npm run changeset` to add a changeset describing what changed and which
package(s) it affects. See https://github.com/changesets/changesets for usage.
