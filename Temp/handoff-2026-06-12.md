## Ce qu'il ne faut PAS faire

- Modifier `tokens/component.json` sans approbation humaine explicite
- Valeurs en dur dans les composants → toujours `var(--agtc-*)`
- Dans les scripts Figma : `hexRgb()` ou token primitif dans un fill → toujours `vFill(tokenSémantique, fallback)`
- Dans les scripts Figma : `.characters` avant `.fontName` sur un TextNode
- Pusher directement sur `main` sans quality gate
- Merger une PR sans approbation humaine
