Règles absoluesAbsolute rules

  • Toujours utiliser <agtc-icon name="…" size="control">Always use <agtc-icon name="…" size="control">
  • Icône sémantique (seule info visible) → label="…" obligatoireSemantic icon (sole visible info) → label="…" required
  • Icône décorative (texte adjacent) → decorative obligatoireDecorative icon (adjacent text) → decorative required
  • Taille en dur : style="width:20px" — utiliser size="control"Hardcoded size: style="width:20px" — use size="control"
  • SVG inline hors <agtc-icon> — pas de contrat d'accessibilitéInline SVG outside <agtc-icon> — no accessibility contract
  • Icône sémantique sans label ni decorativeSemantic icon without label or decorative
