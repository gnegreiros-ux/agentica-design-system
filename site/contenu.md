## Site web
### Hero / About
#### FR 

**Agentica est un système de design conçu pour deux types d'interlocuteurs : les humains qui décident, et les agents IA qui exécutent.**

Là où un système classique documente pour être consulté, Agentica instruite pour être exécuté. Chaque token porte une intention. Chaque composant est un contrat. Chaque règle est assez explicite pour traverser toute l'organisation sans se perdre en chemin — et pour être appliquée correctement par des agents, sans interprétation.
Les humains gardent le dernier mot. Toujours. Les agents observent, détectent, proposent. Les humains approuvent, décident, déploient.
Dans ce modèle, le rôle du designer évolue. Il ne produit plus seulement des artefacts — il devient architecte de décisions reproductibles. (Smashing Magazine / Romina Kavcic)

**Ce qui rend Agentica différent :**

- **Agentique par conception** — les agents lisent les règles, détectent les dérives, proposent des corrections. Les humains valident.
- **Auto-surveillance** — la cohérence entre Figma, les tokens et le code est vérifiée en continu.
- **Stack agnostique** — React, Vue, Angular, systèmes patrimoniaux. Un seul système, partout.
- **Documentation vivante** — elle se met à jour avec le système, automatiquement.
- **Audit à coût marginal** — la conformité est un livrable du système, pas un projet parallèle.
- **Souveraineté numérique** — vos données, vos décisions, vos outils. Aucune dépendance imposée.

#### EN 

**Agentica is a design system built for two kinds of stakeholders: humans who decide, and AI agents who execute.**

Where a traditional system documents to be consulted, Agentica instructs to be acted upon. Every token carries intent. Every component is a contract. Every rule is explicit enough to travel across the organization without getting lost — and to be applied correctly by agents, without interpretation.
Humans always have the final word. Agents observe, detect, propose. Humans approve, decide, deploy.
In this model, the designer's role shifts. They no longer just produce artifacts — they become architects of reproducible decisions. (Smashing Magazine / Romina Kavcic)

**What makes Agentica different:**

- **Agentic by design** — agents read rules, detect drift, propose corrections. Humans validate.
- **Continuous self-monitoring** — consistency between Figma, tokens, and code is verified automatically.
- **Stack agnostic** — React, Vue, Angular, legacy systems. One system, everywhere.
- **Living documentation** — it updates with the system, automatically.
- **Audit at marginal cost** — compliance is a system output, not a side project.
- **Digital sovereignty** — your data, your decisions, your tools. No imposed vendor lock-in.
