Vérifications manuelles requisesRequired manual checks

Ces critères ne peuvent pas être vérifiés automatiquement. Une revue manuelle périodique est recommandée. These criteria cannot be automatically verified. Periodic manual review is recommended.

  • SC 1.4.1
    Usage de la couleurUse of Colour Les erreurs ne sont pas signalées par la couleur seule (texte + icône + couleur).Errors are not conveyed by colour alone (text + icon + colour).
  • SC 1.4.4
    Redimensionnement du texteResize Text Tester à 200% de zoom — pas de perte de contenu ni de fonctionnalité.Test at 200% zoom — no content or functionality loss.
  • SC 1.4.10
    Redistribution (Reflow)Reflow Tester à 320px — pas de défilement horizontal requis.Test at 320px — no horizontal scroll required.
  • SC 1.4.12
    Espacement du texteText Spacing Tester avec line-height ≥ 1.5, letter-spacing ≥ 0.12em, word-spacing ≥ 0.16em.Test with line-height ≥ 1.5, letter-spacing ≥ 0.12em, word-spacing ≥ 0.16em.
  • SC 2.1.1
    Navigation clavierKeyboard Tester Tab / Shift+Tab / Entrée / Espace sur tous les contrôles interactifs.Test Tab / Shift+Tab / Enter / Space on all interactive controls.
  • SC 2.1.2
    Pas de piège clavierNo Keyboard Trap Aucune zone ne bloque la navigation clavier.No zone traps keyboard focus.
  • SC 2.3.1
    Pas de flashThree Flashes Aucun contenu ne clignote plus de 3 fois par seconde.No content flashes more than 3 times per second.
  • SC 2.4.3
    Ordre de focusFocus Order L'ordre de tabulation suit l'ordre visuel logique.Tab order follows the logical visual order.
  • SC 2.4.11
    Focus non masquéFocus Not Obscured Vérifier que le header fixe ne masque pas l'indicateur de focus.Verify fixed header doesn't obscure the focus indicator.
  • SC 4.1.3
    Messages de statutStatus Messages Les messages dynamiques sont annoncés par les lecteurs d'écran (aria-live).Dynamic messages are announced by screen readers (aria-live).

Méthode d'auditAudit method

Cet audit est une analyse statique automatisée du HTML généré. Il vérifie la structure sémantique, les noms accessibles, la hiérarchie des titres, les ratios de contraste des tokens, et les critères WCAG 2.2 détectables sans navigateur. Il ne remplace pas un audit avec lecteur d'écran, ni une vérification de la navigation clavier en conditions réelles. Les critères marqués « vérification manuelle » doivent être testés par un humain. This audit is an automated static analysis of generated HTML. It checks semantic structure, accessible names, heading hierarchy, token contrast ratios, and WCAG 2.2 criteria detectable without a browser. It does not replace a screen reader audit or real-world keyboard navigation testing. Criteria marked "manual check" must be tested by a human.
